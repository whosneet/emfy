# Emfy

A free, open-source macOS viewer for EMF (Enhanced Metafile) files. The point
is not the viewer window — it is Quick Look: spacebar preview in Finder and
rendered thumbnails for `.emf` files, which no free Mac tool currently offers.

**Status: pre-alpha.** Nothing renders yet. The parser and renderer are being
built clean-room from the public [MS-EMF] specification — no code from any
existing EMF implementation.

- Spec: [MS-EMF] — https://learn.microsoft.com/en-us/openspecs/windows_protocols/ms-emf/
- Requires: macOS 14.0 (Sonoma) or later

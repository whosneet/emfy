Support

Get help with Emfy, the free open-source EMF viewer for macOS.

Email hello@neetsingh.com — questions, feedback, and problem reports all welcome. This is the same address as the app's Help → Contact the Developer menu item.
Bug reports GitHub Issues — the most valuable report is a real .emf file that renders wrongly; attach it if you're able to share it.

Which macOS versions does Emfy support?

macOS 14.0 (Sonoma) or later.

Previews or thumbnails aren't showing

Launch the Emfy app once — macOS registers the Quick Look extensions from then on. If Finder still shows generic icons afterwards, log out and back in, and check that Emfy's extensions are enabled in System Settings (search for “Extensions”, then look under Quick Look).

A file renders wrongly or partially

Some files store their drawing content only in the newer EMF+ format; Emfy renders what it can and says so in the viewer — EMF+ decoding is on the v2 list. For anything else that looks wrong, email the file or open an issue and it becomes a test case for the next release.

Where do I get the latest version?

The latest release on GitHub always has the newest notarized Emfy.dmg.

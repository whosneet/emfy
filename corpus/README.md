# corpus/

Committed test files only. A file may live here only if it is **self-generated**
(our own exports from PowerPoint, Excel, LibreOffice Draw, etc.) or
**verifiably CC0/public domain**. Everything else — GPL-sourced test resources,
work-sourced files — belongs in the gitignored `corpus-local/` and is never
committed, screenshotted, or quoted in issues, commits, or docs.
